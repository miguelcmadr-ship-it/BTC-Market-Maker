Set-up:

1. Open command prompt (Win + cmd + Enter)
2. Type cd btc_market_maker
3. Type py -m pip install -r requirements.txt

One-command way to run:

1. Type py main.py


--- To run tests ---

1. Open command prompt (Win + cmd + Enter)
2. Type cd btc_market_maker
3. Type py -m pip install pytest [if not done already]

One-command way to run:

1. Type py -m pytest
